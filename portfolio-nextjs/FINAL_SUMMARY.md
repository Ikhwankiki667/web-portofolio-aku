## 🚀 Cara Menggunakan (3 Langkah Mudah)

### Step 1: Install Dependencies (5 menit)
```bash
cd portfolio-nextjs
npm install
```

### Step 2: Tambahkan File Kamu (10 menit)
```bash

public/images/profile-placeholder.png

# 2. CV PDF

# 3. Project images (optional)
public/images/projects/credit-risk.png
public/images/projects/ds-career.png
```

### Step 3: Run & Test (5 menit)
```bash
npm run dev
```
Buka: http://localhost:3000
